# Group Setup Overview

Setting up your group correctly is essential for smooth trading operations. The setup flow is menu-based and designed to guide you step by step.

## Setup Checklist

- [Initial Setup](./initial-setup.md) - Initialize the bot and create wallets
- [Understanding Wallets](./wallets.md) - Group vs Deployer wallets
- [Configuring Traders](./traders.md) - Add and manage traders
- [Setting Up Payout Wallets](./payout-wallets.md) - Configure distributions
- [Group Settings](./settings.md) - Buy amount and access settings

## Quick Menu Reference

### Owner Actions

| Menu Path | Description |
|---------|-------------|
| Settings -> Traders -> Add Trader | Add a trader |
| Settings -> Traders -> Traders List | View current traders |
| Settings -> Group Call/DCA Amount | Set default buy amount |
| Deployer -> Deploy a Token | Launch a token (owner only) |
| Deployer -> Bundle Wallets | Create bundle wallets |
| Deployer -> Bundle List | View bundle wallets |

### Trader Actions

| Menu Path | Description |
|---------|-------------|
| Settings -> Payouts -> Set Payout Wallet | Set your payout wallet |
| Settings -> Payouts -> Payouts | View payout list |
| Call / DCA / Sell / Take Profit | Trading flows |
| Positions | View and filter positions |

## Understanding Permissions

Sentry Bot uses a trader-based permission system:

- Group Owner: Full control, access to deployer features
- Traders: Can execute trades and set payout wallets
- Members: Can contribute funds and receive distributions

Trading permissions are separate from Telegram admin status.
