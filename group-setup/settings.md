# Group Settings

This guide covers the settings you manage through the Sentry Menu.

## Group Call/DCA Amount (Owner Only)

Sets the default SOL amount used for Call and DCA.

Menu path: Settings -> Group Call/DCA Amount

### Recommended Range

- Start with 0.1 to 0.5 SOL
- Increase as group capital grows

## Traders (Owner Only)

Manage trader permissions here.

Menu path: Settings -> Traders

## Payouts

Manage payout wallets and view payout list.

Menu path: Settings -> Payouts

## My Referrals

Menu path: Settings -> My Refferals

Use this page to view your referral information and links if enabled.

## Best Practices

1. Keep buy amount aligned with wallet balance
2. Review trader list regularly
3. Ensure every trader has a payout wallet

---

Use the Sentry Menu for all settings covered in this guide.
