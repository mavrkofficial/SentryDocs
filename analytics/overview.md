# Analytics and Performance Overview

Use the Analytics buttons in the Sentry Menu to track performance and wallet health.

## Available Buttons

- Balance: Wallet and portfolio snapshot
- Stats: Personal and group performance stats
- Leaderboard: Top traders by performance
- PnL: Group profit and loss report

## Where to Start

1. Open Balance for wallet totals
2. Use Stats to see your performance
3. Check Leaderboard for rankings
4. Review PnL for group-level results

---

Analytics data updates as trades execute and positions close.
