# Deployment Wizard

The Deploy a Token flow is a guided wizard that walks you through every launch step.

## Start the Wizard

1. Open Sentry Menu -> Deployer -> Deploy a Token
2. The wizard opens with editable fields

## Wizard Fields

Use the buttons to fill or update:

- Name (max 32 chars)
- Symbol (ticker)
- Logo (send as photo)
- Website (optional)
- X/Twitter (optional)
- Telegram (optional)
- Bundle (bundle buy settings)
- Dev Buy (first buyer option)

## Bundle Buy Options

If Bundle Buy is enabled, you will choose:

- Funding Method (Deployer-funded or External)
- Bundle Buy Amount

## Dev Buy (First Buyer)

If enabled, you will enter the SOL amount to buy immediately after deployment.

## Deploy

Tap Deploy Token when required fields are complete:

- Name
- Symbol
- Logo

The bot will deploy the token and post the results.

---

Use the wizard buttons to edit fields at any time before deployment.
