# Token Deployment Overview

The Deployer menu is the control center for token launches and bundle wallet management. These features are owner-only.

## Accessing the Deployer Menu

1. Open Sentry Menu -> Deployer
2. Choose the action you want

## Deployer Menu Options

- Deploy a Token
- Harvest Yield
- Deployer Settings
- Bundle Sell
- Bundle Withdraw

---

Deployer tools are designed for owners and require a funded Deployer Wallet.
