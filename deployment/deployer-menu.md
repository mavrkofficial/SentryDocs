# Deployer Menu

The Deployer Menu is owner-only and contains launch and bundle utilities.

## Menu Options

1. Deploy a Token
2. Harvest Yield
3. Deployer Settings
4. Bundle Sell
5. Bundle Withdraw

## Quick Notes

- Deploy a Token launches the deployment wizard
- Deployer Settings lets you create and view bundle wallets
- Bundle Sell handles selling across bundle wallets
- Bundle Withdraw sweeps SOL back to the deployer wallet

---

Always ensure the Deployer Wallet has enough SOL before launching tokens.
