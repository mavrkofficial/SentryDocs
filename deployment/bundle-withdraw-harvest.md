# Bundle Withdraw and Harvest

## Bundle Withdraw

Bundle Withdraw sweeps SOL from all bundle wallets back to the Deployer Wallet.

Menu path: Sentry Menu -> Deployer -> Bundle Withdraw

What happens:

- SOL is transferred from each bundle wallet
- A small amount is kept for rent and fees
- A results report is posted with transaction links

## Harvest Yield

Harvest Yield runs the group's harvest flow for deployed liquidity.

Menu path: Sentry Menu -> Deployer -> Harvest Yield

The bot executes the harvest and posts the results.

---

These actions are owner-only and should be run when needed.
