## Content Guidelines

### Writing Style
- User-friendly language
- Step-by-step instructions
- Menu paths over commands
- Examples and screenshots where helpful

### Each Page Should Include
1. Clear title and introduction
2. Menu flow steps
3. What to expect
4. Common issues
5. Related topics

### Example Page Structure

```markdown
# [Page Title]

[Brief introduction explaining what this feature does]

## How to Use

1. Open Sentry Menu -> [Button]
2. Follow the prompts
3. Confirm the result

## What to Expect

[Describe what users will see/experience]

## Common Issues

- Issue 1: Solution
- Issue 2: Solution

## Related Topics

- Link to related page
```
