# Trading Guide Overview

This section covers trading with the Sentry Menu. All trades are executed through button flows and guided prompts.

## Trading Basics

Authorized traders can trade using the Group Wallet. Every trade is tracked, and profits are distributed automatically.

## Main Trading Buttons

| Button | Description | Guide |
|---------|-------------|------|
| Call | Buy a token | [Calling Tokens](./calling-tokens.md) |
| DCA | Add to a position | [DCA Guide](./dca.md) |
| Sell | Sell entire position | [Selling Guide](./selling-positions.md) |
| Take Profit | Partial sell | [Take Profit Guide](./take-profit.md) |
| Positions | View active positions | [Positions Guide](./viewing-positions.md) |

## Trading Workflow

1. Call -> send the token address
2. Positions -> monitor price, ROI, and PnL
3. DCA / Take Profit / Sell -> manage the position

## Key Concepts

- Entry Price: Price at the time of purchase
- Current Price: Real-time price from DexScreener
- PnL: Profit or loss based on entry vs exit
- ROI: Percentage gain or loss
- Position Size: Amount of tokens held

## Trading Requirements

- You are a trader
- Group Wallet has SOL
- You have set a payout wallet
- Buy amount is configured in Settings

## Best Practices

1. Research tokens before calling
2. Check balances before trading
3. Use take profits strategically
4. Coordinate with other traders
